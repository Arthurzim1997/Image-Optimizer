## Dependencies

Please make sure you have these dependencies first before building.

```
granite
gtk+-3.0
glib-2.0
jpegoptim
optipng
```

## Building

Simply clone this repo, then:

```
$ mkdir build && cd build
$ cmake ..
$ make && sudo make install
$ com.github.gijsgoudzwaard.image-optimizer
```
